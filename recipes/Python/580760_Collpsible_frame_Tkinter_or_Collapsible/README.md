## Collpsible frame for Tkinter or Collapsible panel  
Originally published: 2017-03-04 00:34:09  
Last updated: 2017-03-06 12:40:28  
Author: Miguel Martínez López  
  
Click on button and frame will collapse or expand.

If you want to fit the frame to width of contents, call method "udpate_width()".

Simple API:

- open()
- close()
- toggle()
- update_width()
